# **App Name**: LibFlow

## Core Features:

- Institutional Email Login: Secure login using institutional Google-based emails for authentication.
- User Role Assignment: Assign distinct roles (Administrator or Standard User) upon login to control access to features.
- Visitor Check-in Form: Allows students and faculty to record their reason for visiting the library and their college affiliation.
- Welcome Notification: Displays an instant 'Welcome, to NEU Library!' message upon successful user check-in.
- Admin Dashboard: Provides administrators with a comprehensive overview of library usage statistics, searchable user records, and real-time activity.
- User Management & Blocking: Enables administrators to search for and manage individual user profiles, including the ability to block specific users from accessing the facility.
- AI-Powered Usage Insights Tool: Leverages generative AI to analyze historical visitor data and provide administrators with a tool for insights and trends regarding library utilization by day, week, month, and college.

## Style Guidelines:

- Primary brand color: A deep, professional blue (#223EAA) that signifies stability and knowledge.
- Background color: A very light, subtle bluish-grey (#F0F1F5) providing a calm and clean canvas for content.
- Accent color: A vibrant yet clear blue (#269EDA) for calls to action, highlights, and important information, ensuring strong contrast.
- Headlines and key text elements will use 'Space Grotesk', a sans-serif font, for a modern, tech-forward, and clean aesthetic.
- Body text will utilize 'Inter', a neutral and highly readable sans-serif font, ensuring clarity and legibility for all informational content.
- Employ a set of clear, minimalist, and functional icons to convey actions and information efficiently, especially within the admin dashboard.
- The user check-in interface will be straightforward and uncluttered, while the admin dashboard will feature a structured, card-based layout for data visualization and navigation.
- Implement subtle transition animations for page changes and data updates on the dashboard to provide a smooth and engaging user experience without distractions.
